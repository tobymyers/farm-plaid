Field and Coordinates API Documentation

Get Fields and Coordinates by User ID

Endpoint: /api/get_fields_coordinates/{user_id}

Method: GET

Returns fields and coordinates associated with the specified user ID.

If the user ID is not found, it returns an error message.

Example:

Request: /api/get_fields_coordinates/1

Response:

{
    "fields": [
        {
            "fieldName": "Field 1",
            "area": 10.5,
            "currentCrop": "Wheat",
            "plantDate": "2023-01-01",
            "harvestDate": "2023-06-01",
            "coordinates": [
                {"latitude": 12.34, "longitude": 56.78},
                {"latitude": 23.45, "longitude": 67.89}
            ]
        },
        {
            "fieldName": "Field 2",
            "area": 8.2,
            "currentCrop": "Corn",
            "plantDate": "2023-02-01",
            "harvestDate": "2023-07-01",
            "coordinates": [
                {"latitude": 34.56, "longitude": 78.90},
                {"latitude": 45.67, "longitude": 89.01}
            ]
        }
    ]
}

Replace {user_id} with the actual user ID.
